## 安装vue
1. 查看版本
    $ npm -v
2. 升级 npm
    cnpm install npm -g
3. 升级或安装 cnpm
    npm install cnpm -g
4. 使用vue
    $ cnpm install vue
5. 全局安装 vue-cli
    $ cnpm install --global vue-cli
6. 创建一个基于 webpack 模板的新项目
    $ vue init webpack my-project
